maven checkstyle#

添加maven plugin依赖#

  • configLocation 存放checkstyle的规则配置文件，附录有样例内容
  • SuppressionsLocation 存放屏蔽规则配置文件，附录有样例内容
  • includeTestSourceDirectory 是否检测测试文件夹，建议配置为true
